# 調査の目的


・非現実的な要素を含む「空想」が，夢の内容に与える影響について明らかにする。
 
・空想と夢の関連についての研究の多くは，悪夢などの否定的な要因との関連を調査したものであるため，本研究では，空想の内容全般から，空想が夢に与える影響を明らかにする。
